# Rabiscunho

Olá! Esse é um projeto pessoal de desenvolvimento web, o **Rabiscunho**.

Trata-se de uma aplicação para criar listas de tarefas e escrever anotações. O intuito desse projeto é estudar a arquitetura cliente-servidor e tecnologias fullstack por meio do desenvolvimento de um projeto.

O projeto ainda está em desenvolvimento.

---

## Tecnologias usadas

### Frontend

* HTML
* CSS
* JavaScript
* Fetch API

### Backend

* Node.js
* Express.js
* CORS

### Banco de dados

* SQLite
* sqlite3
