# Rabiscunho
Olá! Esse é um projeto pessoal de desenvolvimento web, o rabiscunho. Trata-se de uma aplicação para criar listas de tarefas e escrever anotações. O intuito desse projeto é estudar a arquitetura cliente-servidor e tecnologias fullstack por meio do desenvolvimento de um projeto.
O projeto ainda está em desenvolvimento.
---

## Tecnologias usadas
- O frontend desse projeto foi desenvolvido usando HTML, CSS, JavaScript e Fetch API;
- O backend usa Node.js, Express.js e CORS;
- O banco de dados usa SQLite e sqlite3
